# Easy-Task

The tool makes task visualization a breeze!

## Usage
For example, let's say you have a file named `example.md`

```markdown 
- [ ] Create project tracker
- [ ] Write Go code
- [X] Build visualization
```

In command line, simply use:

```shell
tct example.md
```
